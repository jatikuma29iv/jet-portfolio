## Configuration
- dotnet
  - omnisharp (needs dotnet core 3.1)
  - sharpenup
  - ale (code analysis)
  - asnycomplete (auto complete)
- git
  - vimagit
- javascript
  - javascript-libraries-syntax
- search
  - file search
    - fzf.vim plugin mapping: `C-P`
  - file search preview colors
    - bat
      - brew install bat
  - text search in file
    - ripgrep with fzf mapping: `<leader>//`
      - brew install reg
- statusline
  - airline
